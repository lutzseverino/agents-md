## Included File

### [`AGENTS.md`](./AGENTS.md)

Lean repo-agnostic guidance for:
- naming tension before continuing
- stopping repeated failed fix loops
- avoiding silent standardization in mixed areas
- routing durable repo knowledge into docs, decisions, or skills

## Use

Copy [AGENTS.md](./AGENTS.md) into the root of a repository when you want always-loaded guidance that helps agents recognize confusion and drift early.

For agent use, reference the file directly rather than assuming the agent will inspect this README on its own.

Examples:

```text
Copy AGENTS.md from lutzseverino/agents-md into this repository.
```

```text
Use the AGENTS.md pattern from https://github.com/lutzseverino/agents-md/blob/main/AGENTS.md
```
